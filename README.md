Volum — Mural Studio Website

A premium multi-page website concept for a contemporary mural and surface art studio.

The project focuses on editorial layout, cinematic pacing, and subtle motion design rather than traditional e-commerce patterns. The visual direction is inspired by modern galleries, architectural studios, and boutique art commerce experiences.

large-scale wall paintings

collectible artworks

The website combines:

subtle scroll interactions

Design Direction

The interface was designed around:

restrained motion
oversized imagery
architectural spacing
muted UI elements
cinematic scrolling rhythm

Instead of aggressive animations, the project uses:

sticky positioning
subtle parallax
slow image transitions
layered spacing systems
Motion Principles

The animation system is intentionally minimal.

Key interactions include:

slow hero image transitions
editorial sticky typography
soft parallax sections
subtle hover scaling
scroll-based pacing

The goal was to create a calm and premium visual atmosphere rather than a highly animated experience.
